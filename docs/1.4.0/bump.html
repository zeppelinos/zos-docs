bump

bump

Usage: bump <version> [options]

bump your project to a new <version>

Options:
--link <stdlib> link to new standard library version
--no-install skip installing stdlib dependencies locally
--push <network> push changes to the specified network after bumping version
-f, --from <from> specify transaction sender address for --push
