init

Usage: init <project> [version]

initialize your ZeppelinOS project. Provide a <project> name and optionally an initial [version] name

Options:
--lib create a standard library instead of an application
--force overwrite existing project if there is one
--link <stdlib> link to a standard library
--no-install skip installing stdlib dependencies locally
--push <network> push changes to the specified network
-f, --from <from> specify transaction sender address for --push
