Ownable

contract Ownable

The Ownable contract has an owner address, and provides basic authorization control functions, this simplifies the implementation of "user permissions". Source: https://raw.githubusercontent.com/OpenZeppelin/openzeppelin-solidity/v2.0.0/contracts/ownership/Ownable.sol.

    function renounceOwnership() public

    Modifiers:
    onlyOwner
